# Kube scaler
Automated load scaler whenever web traffic increases it will automatically trigger Kubernetes pod and scale horizontally

Kubernetes Pod temples:
```python
apiVersion: batch/v1
kind: Job
metadata:
  name: hello
spec:
  template:
    # This is the pod template
    spec:
      containers:
      - name: hello
        image: busybox:1.28
        command: ['sh', '-c', 'echo "Hello, Kubernetes!" && sleep 3600']
      restartPolicy: OnFailure
```
